## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/AavynLabs/aavyn.git
   cd aavyn
   ```

2. **Install dependencies**
   ```bash
   # Using npm
   npm install

   # Using yarn
   yarn install

   # Using pnpm
   pnpm install

   # Using bun
   bun install
   ```

3. **Start the development server**
   ```bash
   # Using npm
   npm run dev

   # Using yarn
   yarn dev

   # Using pnpm
   pnpm dev

   # Using bun
   bun dev
   ```

The application will be available at `http://localhost:5173` by default.
